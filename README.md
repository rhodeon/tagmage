Python script to arrange mp3 files according to their tags.

# USAGE:

## SORTING:
    python3 tagmage.py --sort source_directory destination_directory

### STRUCTURE:
    |----source
      |
      |----artist
         |
         |----album
            |
            |----title.mp3

## EMBEDDING LYRICS:
    python3 tagmage.py --embed working_directory
